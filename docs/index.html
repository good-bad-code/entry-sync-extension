엔트리의 실시간 변수와 리스트를
완전히 대체하는 Chrome 확장 프로그램

Chrome 웹 스토어에서 설치 GitHub

💡 무엇을 하나요?

?!로 시작하는 변수나 리스트를 모든 확장 프로그램 사용자와 실시간으로 동기화합니다. 엔트리의 불안정한 실시간 변수/리스트를 대체할 수 있습니다.

실시간 변수

?!로 시작하는 변수 (예: ?!score)

?!로 시작하는 변수는 이 확장 프로그램을 가지고 있는 모든 유저와 공유됩니다. 즉, 엔트리의 실시간 변수를 대체할 수 있습니다.

실시간 리스트

?!로 시작하는 리스트 (예: ?!items)

?!로 시작하는 리스트는 이 확장 프로그램을 가지고 있는 모든 유저와 공유됩니다. 즉, 엔트리의 실시간 리스트를 대체할 수 있습니다.

🩺 헬스체크 기능

자동 복구

?! 변수 (접두사만 있는 경우)

?!만 있는 변수는 값이 0이 될 경우 자동으로 1로 변경됩니다. 이 기능을 활용해 작품 사용자가 이 확장 프로그램을 가지고 있는지 확인할 수 있습니다.

활용 팁: 작품 시작 시 ?! 변수를 0으로 설정한 후 잠시 기다리면, 확장 프로그램이 설치된 사용자에게는 이 값이 난수로 변경됩니다. 변경되지 않았다면 확장 프로그램이 설치되지 않은 것입니다.

📖 사용 방법

1

Chrome 웹 스토어에서 Entry Sync를 설치합니다.

2

playentry.org에서 엔트리 프로젝트를 엽니다.

3

변수 또는 리스트 이름 앞에 ?!를 붙입니다. (예: ?!score, ?!items)

4

여러 탭이나 다른 사용자와 동일한 프로젝트를 열면 자동으로 실시간 동기화됩니다.

❓ 자주 묻는 질문

엔트리 공식 확장 프로그램인가요?

아닙니다. 개인이 개발한 Chrome 확장 프로그램입니다.

어떤 데이터가 동기화되나요?

?!로 시작하는 변수와 리스트만 동기화됩니다. 일반 변수는 동기화되지 않습니다.

여러 사용자와 동시에 사용할 수 있나요?

네, 확장 프로그램이 설치된 모든 사용자 간에 실시간으로 동기화됩니다.

?! 변수가 0이 되는 이유는 무엇인가요?

Entry가 변수를 초기화할 때 0으로 설정하는 경우가 있습니다. 확장 프로그램이 이를 감지하면 자동으로 1로 변경하여 정상 작동을 유지합니다.

불안정한 실시간 변수는 이제 그만. 안정적인 동기화를 경험해보세요.
