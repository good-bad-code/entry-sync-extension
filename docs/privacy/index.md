# 개인정보처리방침

**최종 업데이트**: 2026년 6월 13일

## 1. 개요

Entry Sync("본 확장 프로그램")는 엔트리(Entry, playentry.org) 작품에서 `?!` 접두사가 붙은 변수와 리스트를 실시간으로 동기화하는 Chrome 확장 프로그램입니다.

본 개인정보처리방침은 확장 프로그램이 수집, 사용, 저장하는 데이터와 그 목적을 설명합니다.

## 2. 수집하는 데이터

### 2.1 수집 항목

| 데이터 | 설명 | 필수 여부 |
|--------|------|-----------|
| **프로젝트 URL** | 사용자가 접속 중인 Entry 작품 페이지의 URL | 필수 (동기화 식별자) |
| **변수명과 값** | Entry 작품 내 `?!` 접두사가 붙은 변수의 이름과 값 | 필수 (동기화 대상) |
| **리스트명과 배열** | Entry 작품 내 `?!` 접두사가 붙은 리스트의 이름과 전체 데이터 | 필수 (동기화 대상) |
| **타임스탬프** | 변수/리스트가 변경된 시각 | 필수 (변경 순서 보장) |

### 2.2 수집하지 않는 데이터

다음 데이터는 **수집하지 않습니다**:

- 이름, 이메일, 전화번호 등 개인 식별 정보
- 비밀번호, 인증 토큰 등 인증 정보
- IP 주소
- 위치 정보 (GPS, 지역 등)
- 건강 정보
- 금융 및 결제 정보
- 방문 기록 (브라우징 히스토리)
- 키 입력, 마우스 움직임 등 사용자 활동 로그
- 쿠키, 추적용 식별자
- 광고 관련 데이터

### 2.3 자동 수집 데이터

확장 프로그램이 자동으로 수집하는 데이터는 오직 Entry 작품 내 `?!` 접두사가 붙은 변수와 리스트의 값뿐입니다. 이는 사용자가 Entry 블록 코딩 과정에서 의도적으로 생성한 데이터입니다.

## 3. 데이터 저장 및 보관

### 3.1 저장 위치

수집된 데이터는 외부 클라우드 데이터베이스에 저장됩니다.

### 3.2 보관 기간

데이터는 사용자가 Entry 작품을 사용하는 기간 동안 유지됩니다. 명시적인 데이터 삭제 기능은 현재 제공되지 않습니다.

### 3.3 보안

- **전송 중 암호화**: 모든 통신은 HTTPS/TLS를 통해 암호화됩니다.
- **저장 중 암호화**: 데이터베이스 수준의 암호화가 적용됩니다.

## 4. 데이터 사용 목적

수집된 데이터는 **오직 하나의 목적**으로만 사용됩니다:

> **Entry 작품의 `?!` 변수/리스트를 여러 탭, 창, 사용자 간 실시간으로 동기화**

## 5. 데이터 공유

### 5.1 제3자 서비스

클라우드 데이터베이스 호스팅 서비스가 데이터 저장 및 실시간 메시지 중계를 위해 사용됩니다.

### 5.2 판매 금지

본 확장 프로그램은 **어떠한 사용자 데이터도 제3자에게 판매하지 않습니다.**

### 5.3 전송 금지

본 확장 프로그램은 **명시된 데이터 동기화 목적 외의 용도로 사용자 데이터를 전송하지 않습니다.**

### 5.4 신용도 평가 금지

본 확장 프로그램은 **신용도 판단이나 대출 목적으로 사용자 데이터를 사용하거나 전송하지 않습니다.**
