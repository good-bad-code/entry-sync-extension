# 개인정보 처리방침

**Entry Sync (엔트리 싱크)**  
최종 업데이트: 2025년 6월 25일

---

## 1. 서비스 소개

Entry Sync는 Chrome 브라우저 확장 프로그램으로, 네이버 커넥트재단의 엔트리(Entry, playentry.org) 플랫폼에서 작품을 만들 때 사용하는 변수와 리스트를 실시간으로 여러 사용자 간에 동기화하는 기능을 제공합니다.

이 확장 프로그램을 사용하면 같은 엔트리 작품을 보고 있는 사용자들이 `?!` 접두사가 붙은 변수와 리스트의 값을 공유하여 협업할 수 있습니다.

---

## 2. 수집하는 데이터

Entry Sync는 다음과 같은 데이터만 수집합니다.

| 수집 항목 | 설명 | 필수 여부 |
|---|---|---|
| 프로젝트 ID | 엔트리 작품 페이지 URL에서 추출한 작품 고유 번호 | 필수 |
| `?!` 변수명과 값 | 사용자가 엔트리 작품 내에서 `?!` 접두사로 공유한 변수와 그 값 | 선택 |
| `?!` 리스트명과 항목들 | 사용자가 엔트리 작품 내에서 `?!` 접두사로 공유한 리스트와 그 항목들 | 선택 |
| 기기 식별자 | Supabase 익명 인증을 위한 무작위 UUID (사용자 식별 불가) | 필수 |

### 수집하지 않는 데이터

Entry Sync는 **절대** 다음과 같은 정보를 수집하지 않습니다.

- 이름, 이메일 주소, 전화번호 등 개인 식별 정보(PII)
- 비밀번호 또는 로그인 정보
- 엔트리 계정 정보
- IP 주소 (단, Supabase 서비스의 표준 연결 로그에 일시적으로 기록될 수 있으나 확장 프로그램이 별도로 수집하지 않음)
- 방문 기록, 검색 기록, 쿠키
- 위치 정보
- 사용자의 개인 파일 또는 시스템 정보
- `?!` 접두사가 **없는** 일반 변수나 리스트
- 엔트리 작품의 블록 코드, 작품 내용, 미디어 파일

---

## 3. 데이터 수집 방법

Entry Sync는 다음과 같은 방식으로 데이터를 수집합니다.

1. **프로젝트 ID 추출**: 사용자가 엔트리(playentry.org) 웹사이트에서 작품 페이지를 열면, 확장 프로그램이 현재 탭의 URL에서 프로젝트 ID를 추출합니다. 예를 들어 `https://playentry.org/project/abc123`에서 `abc123` 부분을 추출합니다.

2. **변수 및 리스트 변경 감지**: 확장 프로그램이 엔트리 작품 실행 환경 내에서 `?!` 접두사로 시작하는 변수와 리스트의 값을 지속적으로 모니터링합니다. 변경이 감지되면 해당 데이터를 Supabase 데이터베이스로 전송합니다.

3. **실시간 동기화**: Supabase Realtime 채널을 통해 같은 프로젝트 ID를 사용하는 다른 사용자들에게 변경된 데이터가 즉시 전달됩니다.

---

## 4. 데이터 저장 및 보안

### 저장 위치

수집된 데이터는 **Supabase Inc.** 가 제공하는 클라우드 데이터베이스에 저장됩니다. Supabase는 AWS 인프라를 기반으로 운영되며, 데이터는 미국에 위치한 서버에 저장됩니다.

### 저장 형식

데이터는 `프로젝트ID:변수명`을 키로, 변숫값을 값으로 하는 형태로 저장됩니다. 예를 들어 `abc123:?!점수`라는 키에 `100`이라는 값이 저장됩니다.

### 보안 조치

- **RLS (Row Level Security)**: Supabase 데이터베이스는 Row Level Security 정책으로 보호되어, 올바른 프로젝트 ID를 가진 요청만 해당 프로젝트의 데이터에 접근할 수 있습니다.
- **익명 인증**: Supabase Auth의 익명 로그인 방식을 사용하며, 개인을 식별할 수 있는 인증 정보는 저장하지 않습니다.
- **전송 암호화**: 모든 데이터는 HTTPS/WSS 프로토콜을 통해 암호화되어 전송됩니다.
- **데이터 유지 기간**: 동기화 데이터는 서비스 운영 기간 동안 유지되며, 확장 프로그램이 데이터를 삭제할 수 있는 기능을 제공합니다.

## 5. 데이터 공유

Entry Sync의 핵심 기능은 **데이터 공유** 자체입니다.

- **공유 대상**: 현재 **같은 엔트리 작품**을 보고 있는 다른 Entry Sync 사용자들
- **공유 범위**: 사용자가 `?!` 접두사를 붙여 명시적으로 공유한 변수와 리스트만 공유됩니다.
- **제3자 공유 없음**: 수집된 데이터를 마케팅, 광고, 분석 목적으로 제3자와 공유하거나 판매하지 않습니다.
- **데이터 분석 목적 없음**: 사용자 활동을 추적하거나 분석하지 않습니다.

## 6. 사용자 권리

사용자는 다음과 같은 권리를 가집니다.

1. **데이터 수집 중단**: 확장 프로그램을 제거하거나 비활성화하면 더 이상 데이터가 수집되지 않습니다.
2. **데이터 수집 방지**: 엔트리 작품 내에서 `?!` 접두사를 사용하지 않으면 데이터가 수집되지 않습니다.
3. **데이터 삭제 요청**: Supabase 데이터베이스에 저장된 데이터의 삭제를 요청할 수 있습니다. GitHub Issues를 통해 요청해 주시면 확인 후 처리해 드립니다.
4. **데이터 열람**: 동기화되는 모든 데이터는 확장 프로그램 팝업을 통해 언제든지 확인할 수 있습니다.

## 7. 제3자 서비스

Entry Sync는 다음과 같은 제3자 서비스를 사용합니다.

| 서비스 | 목적 | 데이터 전송 범위 | 개인정보 처리방침 |
|---|---|---|---|
| **Supabase** (supabase.com) | 데이터베이스 저장 및 실시간 동기화 (WebSocket) | 프로젝트 ID, `?!` 변수/리스트 데이터, 익명 인증 토큰, 기기 UUID | [Supabase Privacy Policy](https://supabase.com/privacy) |
| **Google Fonts** (fonts.googleapis.com) | 확장 프로그램 UI의 Inter 폰트 로드 | IP 주소 (브라우저가 표준 HTTP 요청 시 전송) | [Google Privacy Policy](https://policies.google.com/privacy) |
| **playentry.org** | 엔트리 작품 실행 환경 제공 | 없음 (확장 프로그램이 playentry.org 페이지에서 읽기 전용으로 변수 값을 읽음) | [엔트리 개인정보처리방침](https://playentry.org/privacy) |
